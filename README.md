## Tic Tac Toe

This is a simple tic tac toe game that can be played on the command line as long as your terminal supports ASCII character.

### Visuals

```
    a     b     c
       |     |
1   -  |  -  |  -
  _____|_____|_____
       |     |
2   -  |  -  |  -
  _____|_____|_____
       |     |
3   -  |  -  |  -
       |     |
```
